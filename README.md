# BrewMetrics Coffee Dashboard

## Project Overview 
BrewMetrics Coffee Dashboard is a data visualization and reporting project built to analyze coffee sales performance, customer trends, product demand, and operational insights. The dashboard transforms raw coffee shop data into interactive business intelligence that supports decision-making.
The project was created using Microsoft Excel and focuses on presenting key metrics in a clean, easy-to-understand format.

## Project Objectives

The BrewMetrics Coffee Dashboard aims to:

- Track total sales and revenue performance
- Monitor top-selling coffee products
- Analyze customer purchasing trends
- Compare sales performance across different periods
- Identify peak business hours and high-performing product categories
- Provide a clear visual summary of overall business performance


## Data Source

This project uses a coffee sales dataset obtained from a public GitHub repository. The dataset includes transactional order data, customer records, and product information used for sales analysis and dashboard visualization in Microsoft Excel.

Source:
[Sales Dataset](https://github.com/mochen862/excel-project-coffee-sales/blob/main/coffeeOrdersProject.xlsx)

## Dataset Features

| Column Name | Description |
|---|---|
| Order ID | Unique identifier for each transaction |
| Order Date | Date the order was placed |
| Customer ID | Unique identifier for each customer |
| Customer Name | Name of the customer |
| Product | Coffee product purchased |
| Quantity | Number of items purchased |
| Unit Price | Price per product unit |
| Sales | Total revenue generated from the transaction |
| Country | Customer location |

## Data Visualizations

The BrewMetrics dashboard uses multiple Excel visualizations to transform raw data into actionable insights:

- **Sales Trend Analysis:** Line chart displaying revenue performance over time  
- **Product Performance:** Bar chart highlighting top-selling coffee products  
- **Category Breakdown:** Pie chart showing distribution of sales across product categories  
- **Key Performance Indicators (KPIs):** Summary cards for total revenue, total orders, and average order value  
- **Interactive Filters:** Slicers used to filter data by date, product type, and location for deeper analysis
